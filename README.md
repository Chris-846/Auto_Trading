# 삼성전자 자동 매매 프로그램 (모의투자용)

이 프로젝트는 한국투자증권 Open API를 활용한  
삼성전자(005930) 모의투자 자동 매매 시스템이다.
REST API만 사용하며, 09:10부터 15:30까지 거래 시간 동안 현재가를 조회하고,  
현재가 기준 ±2,000원 가격으로 매수·매도 주문을 자동 실행한다.

---

## 주요 기능

- 안전한 환경변수 기반 API 인증키 관리  
- 당일 토큰 캐시로 토큰 재사용 최소화  
- 삼성전자 현재가 실시간 조회 (폴링 방식)  
- 계좌 잔고 및 보유종목 조회  
- 매수·매도 주문 자동 실행과 실행 결과 확인  
- 09:10 ~ 15:30 거래 시간 외 자동 중지  
- 제한된 API 호출량에 최적화된 보수적 폴링 및 재시도 포함  
- 상세 로그 기록으로 상황 모니터링 가능

---

## 필수 환경 변수 설정

안전하게 모의투자 API를 쓰려면 아래 환경변수를 설정해야 한다.  

| 변수명              | 설명                          |
|-------------------|-----------------------------|
| GH_ACCOUNT        | 본인 한국투자증권 계좌 ID          |
| GH_APPKEY         | 발급받은 앱 키                     |
| GH_APPSECRET      | 발급받은 앱 시크릿                  |
| GH_PAPER_ACCOUNT  | 모의투자용 계좌번호                  |
| GH_API_DOMAIN     | API 서버 주소 (모의투자 기본값 있음)  |
| GH_USER_AGENT     | HTTP 요청 시 사용할 User-Agent 문자열  |

환경변수는 `.env` 파일이나 운영체제 설정에서 관리할 수 있다.

## 설치 및 실행

1. Python 3.11 이상 설치  
2. 저장소 클론 후 프로젝트 폴더로 이동
3. uv 패키지 매니저 설치 및 의존성 설치
(설치 방법은 uv 공식 사이트 참고)
4. 환경변수 설정
5. 프로그램 실행
